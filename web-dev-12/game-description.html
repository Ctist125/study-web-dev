게임규칙

설명

  • 해당 게임은 주사위로 진행되는 게임입니다.
  • 1인당 1만원을 가지고 시작하며, 먼저 돈을 모두 따는 사람이 이기게 됩니다.
  • 동점이 나올 경우 딜러의 승리로 처리됩니다.

규칙

  • 각각 주사위를 하나씩 가지고 있고, 공용 주사위가 하나 있습니다.
  • 주사위를 동시에 굴리게 되고 더 큰 수를 보유한 사람이 이기게 됩니다.
  • 승리를 하게 되면, 배팅한 금액의 2배를 얻게되며, 패배를 하게 될 경우 배팅한 금액을 모두 잃게 됩니다.
